Install TorchBrain
==================

**torch_brain** is available for Python >= 3.10.
To install the latest stable release, run::

    pip install torch torch_brain

To install the latest (*unstable*) development version, run::

    pip install torch git+https://github.com/neuro-galaxy/torch_brain


.. tip::

   You can skip installing ``torch`` if you only wish to work with
   :obj:`torch_brain.data` and :obj:`torch_brain.pipeline` related modules,
   e.g. when focusing on pre-processing datasets.
