Using Benchmarks
================

Guides for standardized evaluation protocols built on prepared brainsets. These
pages focus on **Dataset** usage, **Splits**, and **Sampling Intervals** — not
brainset preparation.

Before starting, see :doc:`../brainsets/getting_started/index` to prepare the
underlying data with the ``brainsets`` CLI.
